Powerful AI Features

Advanced machine learning models working together to keep your browsing safe

Text Classification

Automatically scans and classifies text content using the Xenova/toxic-bert model with 98.4% accuracy. Replaces inappropriate content with "***" in real-time.

  • Real-time sentence analysis
  • Configurable strictness levels
  • Context menu integration

Image Classification

Zero-shot image classification using CLIP technology. Automatically detects and filters inappropriate images with 96.5% accuracy.

  • Dynamic content detection
  • Multiple safety categories
  • Automatic image extraction

WebGPU Acceleration

Hardware-accelerated inference for lightning-fast performance. Process thousands of sentences per second with minimal resource usage.

  • Hardware acceleration
  • Efficient memory management
  • Background processing

Smart Controls

Fine-tune your filtering experience with independent controls for text and image filtering, plus adjustable strictness levels.

  • Independent filter toggles
  • Adjustable sensitivity
  • Real-time statistics
